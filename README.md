### UI  
Currency converter shows how much Polish Zloty is worth in other currencies. You can convert Polish Zloty to:
- Euro (EUR)
- Dollar (USD)
- British Pound (GBP)
- Swiss Frank (CHF)
- Czech Koruna (CZK)
